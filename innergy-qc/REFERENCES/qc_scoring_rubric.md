# INNERY QC Scoring Rubric

**Autoresearch Framework — Millwork Estimating QC**

**Goal:** Score each QC run against this checklist. Target: 95%+ three times in a row before a skill version is "locked."

## QC Scoring Checklist

Each item is worth points. Total = 100%.

### Section A — Dimension Accuracy (40 points)

- A1 (10 pts) All cabinet X dimensions match INNERGY within 0.5" tolerance
- A2 (10 pts) All cabinet Y (height) dimensions match INNERGY within 0.5" tolerance
- A3 (10 pts) All cabinet Z (depth) dimensions match INNERGY within 0.5" tolerance
- A4 (10 pts) No systematic height discrepancies > 1"

### Section B — Completeness (30 points)

- B1 (10 pts) No missing cabinets — every cabinet on drawings has an INNERGY line item
- B2 (10 pts) No extra items — every INNERGY line item has drawing evidence
- B3 (10 pts) No mislabeled items (countertops labeled as floating shelves, etc.)

### Section C — Material Codes (20 points)

- C1 (10 pts) All PL-/SS-/WD-/AT- material codes matched
- C2 (10 pts) Finish schedule aligned (Caesarstone, FENIX, Walnut, PT-01/PT-02, LA-02)

### Section D — Quantities (10 points)

- D1 (5 pts) Cabinet counts match per suite
- D2 (5 pts) Quantity (QTY) field matches drawing count

## Scoring

| Score | Grade |
|-------|-------|
| 95-100% | 🟢 Excellent |
| 85-94% | 🟡 Good |
| 70-84% | 🟠 Needs Work |
| <70% | 🔴 Poor |

## Scoring Output Format

For each QC run, output:

```
QC Score: XX%
Grade: [Grade]
Failed items: [List]
Autoresearch log: Round N → Score X% → [changes]
```
